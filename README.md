# Skills

Personal skills for AI coding agents.

## Usage

```bash
npx skills@latest add npalladium/skills
```

## Structure

```
skills/
  <skill-name>/
    SKILL.md          # Required: frontmatter + instructions
    references/       # Optional: supporting docs
    scripts/          # Optional: automation helpers
```
